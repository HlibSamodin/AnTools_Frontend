# AnTools

A modern web application for discovering and managing online tools with a user-friendly interface. AnTools helps users browse, search, and organize various online utilities in one convenient location.
- **User Accounts**: Login and Sign Up functionality for personalized experiences
- **Responsive Design**: Built with modern CSS and responsive web design principles
- **Collections**: Users can save and organize their favorite tools

## Project Structure

```
AnTools_Project/
├── index.html          # Main HTML file
├── README.md           # Project documentation
├── css/
│   ├── reset.css       # CSS reset/normalize styles
│   └── style.css       # Main stylesheet
└── images/
    └── main-pictures/  # Image assets (logos, etc.)
```

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Styling and layout with flexbox
- **Fonts**: Google Fonts (Poppins, Roboto)
- **Normalize**: Modern-normalize for cross-browser consistency

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the application using the header menu

## File Structure Details

- **index.html**: Main entry point containing the header with navigation menu, search section, and login/signup buttons
- **css/reset.css**: Base styling reset for consistent cross-browser appearance
- **css/style.css**: Custom styles for the AnTools design system
- **images/main-pictures/**: Logo and other visual assets

## Navigation

- **Home**: Return to the main page
- **Categories**: Browse tools by category
- **My Collections**: Access saved tool collections
- **Blog**: Read articles and updates
- **Login/Sign Up**: User authentication
